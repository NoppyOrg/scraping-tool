# scraping-tool
スクレーピングのサンプルコード


# 使い方
## セットアップ
### (1) node.jsのセットアップ
[Node.js — Download Node.js®](https://nodejs.org/en/download)を参照しインストール

### (2) git clone
```sh
git clone https://github.com/NoppyOrg/scraping-tool.git
```

### (3) JavaScriptパッケージインストール
- srcに移動
```sh
cd src
```
- パッケージインストール
```sh
npm install
```

## 実行
- build
```shell
npm run build
```
- 実行
```shell
npm run start
```
